# Sayna Examples

Example implementations demonstrating how to build real-time voice applications with [Sayna](https://github.com/saynaai/sayna).

## About Sayna

Sayna is a real-time voice processing platform that enables Speech-to-Text (STT), Text-to-Speech (TTS), and multi-participant voice rooms via LiveKit integration. Build voice-enabled AI agents, phone systems, transcription services, and interactive voice applications.

For the main Sayna server and documentation, visit the [Sayna repository](https://github.com/saynaai/sayna).

## SDKs

Examples in this repository use the official Sayna SDKs:

| SDK | Package | Platform | Use Case |
|-----|---------|----------|----------|
| [JavaScript SDK](https://github.com/saynaai/saysdk/tree/main/js-sdk) | `@sayna-ai/js-sdk` | Browser | Client-side voice rooms |
| [Node.js SDK](https://github.com/saynaai/saysdk/tree/main/node-sdk) | `@sayna-ai/node-sdk` | Node.js 18+ | Server-side voice processing |
| [Python SDK](https://github.com/saynaai/saysdk/tree/main/python-sdk) | `sayna-client` | Python 3.9+ | Async voice applications |

## Getting Started

1. Clone this repository
2. Choose an example that matches your use case
3. Follow the example's README for setup instructions
4. Refer to the [Sayna documentation](https://docs.sayna.ai) for API details
